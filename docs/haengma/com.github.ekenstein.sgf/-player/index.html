Provides information about a player that is part of the game.

Parameters

name

The name of the player.

rank

The rank of the player. For Go the following format is recommended:

  • "..k" or "..kyu" for kyu ranks and

  • "..d" or "..dan" for dan ranks.

Go servers may want to add '?' for an uncertain rating and '*' for an established rating.

team

Provides the name of the player team, if game was part of a team-match (e.g. China-Japan Supermatch)
